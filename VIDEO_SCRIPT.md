## 第二部分：功能演示 (0:45 - 1:45)

**【画面】**
*   **录屏**：展示系统 Web 界面（清爽的仪表盘）。
*   **操作**：点击“上传任务”，一口气拖入 5 个 3MF 文件。
*   **分屏特效**：左边是 Web 界面显示任务“排队中” -> “上传中”，右边是打印机摄像头的实时画面（或者实拍打印机屏幕）。
*   **关键时刻**：看到打印机屏幕亮起，自动开始下载并打印，无需人工干预。
*   **多机演示**（如果有）：展示两台打印机，任务自动分配给了空闲的那一台。

**【口播】**
“看，这就是它的后台界面。部署在 NAS 或者电脑上，随时随地都能访问。”
“操作非常简单，只需要把切好片的 3mf 文件批量拖进来。系统会自动计算文件的 MD5，防止重复上传。”
“最核心的功能来了：**智能调度**。系统会实时监控打印机的状态，一旦机器空闲，它就会自动把下一个任务‘喂’给打印机。甚至支持多台打印机负载均衡，谁有空谁就干活，真正实现 24 小时不停机。”

## 第三部分：部署教程 (硬核干货) (1:45 - 3:15)

**【画面】**
*   **录屏**：威联通 Container Station 界面。
*   **步骤 1**：展示上传代码文件夹到 File Station。
*   **步骤 2**：打开 Container Station，创建 Application。
*   **步骤 3**：复制粘贴 YAML 代码（重点高亮修改 IP 和 Access Code 的地方）。
*   **步骤 4**：点击 Create，等待绿色图标亮起。

**【口播】**
“为了让大家都能用上，我把这套系统做成了 Docker 容器。不管你是 Windows、Mac 还是群晖、威联通 NAS，都能一键部署。”
“这里以威联通 NAS 为例，只要三步：”
1.  “把代码包上传到 NAS。”
2.  “打开 Container Station 创建应用。”
3.  “填入你的打印机 IP 和访问码，粘贴配置代码。”
“搞定！不需要懂代码，只要会复制粘贴就行。具体的代码和配置文件，我都放在简介里的开源地址了。”
